Блок 3: Сортировка услуг
TC-ID	Название	Шаги	Ожидаемый результат
TC-SORT-01	Сортировка по названию (А-Я)	1. Выбрать сортировку «По названию (А-Я)»
2. Применить	Услуги отсортированы в алфавитном порядке по полю Title (от А до Я).
TC-SORT-02	Сортировка по названию (Я-А)	1. Выбрать сортировку «По названию (Я-А)»
2. Применить	Услуги отсортированы в обратном алфавитном порядке.
TC-SORT-03	Сортировка по цене (возрастание)	1. Выбрать сортировку «Сначала дешёвые»
2. Применить	Услуги отсортированы по полю Cost от меньшего к большему.
TC-SORT-04	Сортировка по цене (убывание)	1. Выбрать сортировку «Сначала дорогие»
2. Применить	Услуги отсортированы по полю Cost от большего к меньшему.
TC-SORT-05	Сортировка по длительности (возрастание)	1. Выбрать сортировку «По длительности (сначала короткие)»
2. Применить	Услуги отсортированы по полю Durationinservice (в минутах) по возрастанию.
TC-SORT-06	Сортировка по длительности (убывание)	1. Выбрать сортировку «По длительности (сначала длинные)»
2. Применить	Услуги отсортированы по убыванию длительности.
TC-SORT-07	Сортировка по скидке	1. Выбрать сортировку «По размеру скидки»
2. Применить	Услуги отсортированы по полю Discount (от большего к меньшему или наоборот – согласно реализации).
TC-SORT-08	Комбинированная сортировка (выбор другого поля после уже применённой сортировки)	1. Отсортировать по названию
2. Затем отсортировать по цене	Сортировка применяется по последнему выбранному полю (предыдущая сортировка сбрасывается).
