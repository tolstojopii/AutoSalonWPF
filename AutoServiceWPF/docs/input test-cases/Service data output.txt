Блок 1: Вывод данных об услугах
TC-ID	Название	Шаги	Ожидаемый результат
TC-DISPLAY-01	Вывод всех услуг	1. Открыть страницу «Услуги»
2. Дождаться загрузки таблицы	В таблице отображаются все услуги из БД (количество строк равно числу записей в таблице service).
TC-DISPLAY-02	Проверка корректности данных в строке услуги	1. Выбрать любую услугу в таблице
2. Сравнить отображаемые поля (название, цена, длительность, скидка) с данными в БД	Все поля соответствуют значениям в базе данных.
TC-DISPLAY-03	Отображение скидки	1. В БД для услуги задать скидку 15%
2. Открыть страницу услуг	Услуга отображается с указанием скидки (например, зачёркнутая старая цена и новая со скидкой или иконка «скидка»).
