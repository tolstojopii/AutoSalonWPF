Блок 2: Поиск услуг
TC-ID	Название	Шаги	Ожидаемый результат
TC-SEARCH-01	Поиск по полному названию	1. В поле поиска ввести точное название существующей услуги (например, «Замена масла»)
2. Нажать кнопку «Найти» (или поиск срабатывает автоматически)	В таблице отображается только одна услуга с указанным названием.
TC-SEARCH-02	Поиск по части названия	1. Ввести часть названия (например, «масло»)
2. Запустить поиск	Отображаются все услуги, содержащие слово «масло» в названии (без учёта регистра, если реализовано).
TC-SEARCH-03	Поиск с пустой строкой	1. Очистить поле поиска
2. Запустить поиск	Отображается полный список услуг (как при открытии страницы).
TC-SEARCH-04	Поиск несуществующей услуги	1. Ввести заведомо отсутствующее название (например, «xyz123»)
2. Запустить поиск	Таблица пуста, выводится сообщение «Ничего не найдено» или просто пустая таблица.
TC-SEARCH-05	Поиск с учётом регистра (если реализовано)	1. Ввести название услуги в нижнем регистре (например, «замена масла»)
2. Запустить поиск	Находятся услуги независимо от регистра (должны отображаться). Если регистр важен – должно быть точное совпадение (уточнить по требованиям).
